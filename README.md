## 📌 Sobre o Projeto

Este projeto consiste na recriação da tela principal (Dashboard) de uma aplicação Fintech chamada **FinSmart**. O objetivo principal foi aplicar os conceitos de marcação semântica, estilização em cascata e design responsivo (Mobile First), garantindo uma interface moderna, atraente e funcional sem a necessidade de lógicas em JavaScript.

O projeto segue rigorosamente a separação de responsabilidades, mantendo a estrutura no arquivo `.html` e as customizações finas no arquivo `.css`.

## ✨ Funcionalidades e Diferenciais

- **Design Dark Mode:** Interface imersiva e moderna, focada no conforto visual do usuário, com detalhes em magenta (Brand Pink).
- **Layout Responsivo:** Adaptação perfeita para dispositivos móveis, tablets e desktops (Mobile First).
- **Gráficos em SVG Nativo:** Implementação de gráficos de evolução (linhas) e distribuição (rosca) utilizando apenas matemática SVG e marcação HTML, cumprindo a regra de não utilizar JavaScript.
- **Scrollbar Customizada:** Barras de rolagem refinadas via CSS para tabelas em resoluções menores, mantendo a elegância do layout.
- **Estrutura Semântica:** Uso correto de tags HTML5 (`<main>`, `<aside>`, `<nav>`, etc.) para melhor acessibilidade e organização do código.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica e importação de assets (fontes, ícones SVG embutidos).
- **CSS3:** Estilização global, animações e refinamentos de UI (arquitetado no arquivo `style.css`).
- **Tailwind CSS:** Framework utilitário carregado via CDN para construção ágil do layout em grid/flexbox e tipografia.
- **Google Fonts (Inter):** Tipografia limpa e profissional, ideal para painéis de dados.
